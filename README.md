# 🚀 Minimalist Habit Tracker

A clean, aesthetic glassmorphism habit tracker built with vanilla HTML5, CSS3, and JavaScript (ES6+). This application helps users track their daily tasks with automated progress statistics and data persistence.

🔗 **[Live Demo: View the Habit Tracker Here](https://suroj-mrzn.github.io/Habit-tracker/)** ---

## ✨ Features

* **🎨 Glassmorphism UI**
* **📈 Real-Time Progress Engine** 
* **💾 Persistent Local Storage**
* **🧹 Full CRUD Lifecycle**
